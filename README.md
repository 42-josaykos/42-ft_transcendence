# ft_transcendence

## Development

### Backend

```bash
$ cd backend
$ npm install
$ npm run dev
```

### Frontend

```bash
$ cd Frontend
$ npm install
$ npm run dev
```

A Makefile is available to easily launch dev environments locally.
